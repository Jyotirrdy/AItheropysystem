# AI Therapy System

A lightweight AI therapy support prototype implemented in Python.

## Features
- Simple emotional cue detection (anxious, sad, angry, stressed)
- Crisis-language detection with immediate escalation guidance
- Supportive reflective prompts
- Interactive command-line chat

## Run
```bash
python ai_therapy_system.py
```

## Test
```bash
python -m pytest -q
```

## Important Disclaimer
This project is **not** a replacement for licensed mental-health professionals.
If someone is in immediate danger, call emergency services.
